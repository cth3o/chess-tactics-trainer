## Description

This is a chess tactics trainer. It is a software for lichess game analysis and tactics training.

## Getting Started

### Dependencies

* Python >= 3.8

### Installing

```bash
pip install -r requirements.txt -r requirements-dev.txt
```

### Executing program

```bash
python src/main.py
```

### Launch tests

```bash
pytest tests/
```
